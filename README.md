# タッチタイピング練習

手元を見ずに打つ「ブラインドタッチ」を練習するためのシンプルな Web アプリです。
画面上のキーボードが次に押すキーと担当指をハイライトしてくれるので、キーボードを見ずに練習できます。

**▶ 今すぐ試す: https://asahi73.github.io/typing-practice/**

## 特徴

- ⌨️ **次キー＆担当指ハイライト** — 画面上のキーボードで次に押すキーが光り、どの指で打つかを色で表示
- 📊 **リアルタイム計測** — WPM（1分あたりの入力速度）・正確率・経過時間・ミス数を表示
- 🏆 **ベストスコア保存** — モードごとの自己ベストをブラウザ（localStorage）に保存
- 🚫 **インストール不要** — ブラウザだけで動作。アカウント登録もなし

## 練習モード

| モード | 内容 |
| --- | --- |
| **ホームポジション** | `asdf jkl;` から全アルファベットまで、段階を選んで基礎を反復 |
| **小指特訓** | 最も弱い小指の担当キー（`a` `q` `z` `p` `;` `/`）だけを集中練習 |
| **英単語・英文** | 実用的な英文を打ってスピードと正確さを鍛える |
| **日本語ローマ字** | ひらがな文をローマ字入力。`し`=shi/si など複数の打ち方を受理し、促音・拗音・撥音にも対応 |

## 使い方

1. 上部のタブで練習モードを選ぶ
2. 表示された文字を、**手元を見ずに**画面のキーガイドを頼りに打つ
3. 打ち終わると結果（WPM・正確率）が表示される。`Enter` で次のお題へ
4. 途中でやめたいときは「中止」、別の問題にしたいときは「別のお題に変える」

## 技術構成

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vite.dev/)（ビルド / 開発サーバー）
- 状態管理・ローマ字判定などはすべて自前実装（外部ランタイム依存なし）
- ホスティング: GitHub Pages（`main` への push で自動デプロイ）

## ローカルで動かす

```bash
git clone https://github.com/Asahi73/typing-practice.git
cd typing-practice
npm install
npm run dev        # http://localhost:5173/ を開く
```

その他のコマンド:

```bash
npm run build      # 本番ビルド（型チェック込み）
npm run preview    # ビルド結果をローカルでプレビュー
```

ローマ字判定ロジックの簡易テスト:

```bash
node --experimental-strip-types scripts/romaji_check.ts
```
